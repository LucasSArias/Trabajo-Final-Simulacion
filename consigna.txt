Simulador de Balance Energético Residencial/Industrial Basado en Datos Reales
Objetivo: Desarrollar un modelo de simulación de Δt fijo (horario) para evaluar la eficiencia de un sistema de energía renovable (panel solar + batería) mediante el uso de series temporales históricas.
Descripción de la idea: El proyecto consiste en utilizar el dataset de Open Power System Data (OPSD), específicamente la serie de generación solar y carga eléctrica, para simular el comportamiento de un sistema de energía autosustentable a pequeña escala.
El modelo utilizará un paso de tiempo fijo (Δt = 1 hora) para procesar la siguiente lógica:
Entrada: Se extraen datos de generación solar (oferta) y carga eléctrica (demanda) de una región geográfica específica.
Procesamiento: En cada unidad de tiempo, el simulador calcula el balance neto: Balance = Generación - Demanda.
Estado: Se actualiza el nivel de carga de una batería virtual:
Si el balance es positivo, se almacena el excedente (carga de batería).
Si el balance es negativo, se cubre el déficit utilizando la energía almacenada (descarga de batería).
Salida: El sistema arrojará métricas de eficiencia, como horas de autosuficiencia, excedentes desperdiciados y momentos críticos de falta de energía.
Utilidad: Esta herramienta permite comprender la intermitencia de las energías renovables y dimensionar correctamente qué capacidad de batería se necesita para que un hogar pueda depender totalmente de sus paneles solares basándose en condiciones meteorológicas y de consumo reales.

Estaría bueno también considerar el costo de las baterías, los paneles solares y el costo del consumo de la red eléctrica, para poder estimar si la inversión inicial es rentable, el tiempo que se tarda en amortizar, etc.
